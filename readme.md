# Go (_Golang_) code examples

## fmt_verbs
Uses some format verbs to display a number.

- Deciamal
- Binary
- Char
- Octal
- Hex (_3 formats_)

## for_and_format_examples
Uses a for to generate a numerical secuences and display every generated number in three formats.

- Decimal `%d`
- Binary `%b`
- Hex `%#X`
- Char `%q`
